Disputes

Your starting point for dispute onboarding: learn when to report an issue, how resolution works, and what to prepare before you file.

Resolve transaction issues quickly

Disputes help you report incorrect, missing, or unauthorized transaction outcomes. Start from a specific transaction so operations can review faster.

Why disputes happen

Delayed settlement, wrong beneficiary details, duplicate charges, or an unexpected debit.

How it works

  1. Report issue
  2. Review and provide evidence
  3. Track resolution updates

Transaction reference, beneficiary details, timeline, and supporting files.
